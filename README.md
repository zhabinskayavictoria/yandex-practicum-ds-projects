# Проекты по Data Science — Яндекс.Практикум 2020

В этом репозитории собраны проекты, выполненные мной в рамках обучения на курсе "Специалист по Data Science" в Яндекс.Практикуме.


### 1. [Рекомендация тарифов](ml-classification)

**Задача:** Построить модель классификации, которая по поведенческим данным пользователя (количество звонков, интернет-трафик и т.д.) сможет рекомендовать ему один из двух тарифов мобильной связи: «Смарт» или «Ультра».

Изучены данные, обучены и сравнены три модели: дерево решений, случайный лес и логистическая регрессия. Модели оценивались по метрике accuracy. 

`pandas`, `scikit-learn` (`train_test_split`, `DecisionTreeClassifier`, `RandomForestClassifier`, `LogisticRegression`, `accuracy_score`).

### 2. [Прогнозирование оттока клиентов банка](ml-classification-imbalanced)

**Задача:** Спрогнозировать, уйдёт ли клиент из банка в ближайшее время.

Основная сложность проекта — сильный дисбаланс классов (уходящих клиентов значительно меньше). Для борьбы с дисбалансом использован метод upsampling (увеличение выборки редкого класса). Модели оценивались по F1-мере и ROC-AUC, которые лучше подходят для несбалансированных данных.

`pandas`, `scikit-learn` (`train_test_split`, `StandardScaler`, `RandomForestClassifier`, `LogisticRegression`), `matplotlib`.

### 3. [Классификация токсичных комментариев](nlp-toxic-comments) 

**Задача:** Разработать модель для интернет-магазина, которая будет обнаруживать и отправлять на модерацию токсичные комментарии пользователей.

Для борьбы с дисбалансом классов протестированы два подхода: изменение весов классов и уменьшение выборки (downsampling). Тексты комментариев были лемматизированы и преобразованы в числовые признаки с помощью TF-IDF. Для поиска оптимальных гиперпараметров использован `GridSearchCV`. Обучены и сравнены модели логистической регрессии и CatBoost.

`pandas`, `nltk`, `pymystem3`, `scikit-learn` (`TfidfVectorizer`, `GridSearchCV`, `LogisticRegression`, `DecisionTreeClassifier`), `catboost` `matplotlib`.

### 4. [Прогнозирование температуры плавки стали](ml-regression-temperature)

**Задача:** Построить модель машинного обучения, которая предсказывает финальную температуру стали на этапе ее обработки в ковше. Основная цель — помочь металлургическому комбинату оптимизировать производственные расходы за счет снижения потребления электроэнергии.

Объединены и предобработаны данные из нескольких источников (электроды, подача сыпучих материалов и проволоки, продувка газом, замеры температуры). Проведен анализ корреляций, устранены аномалии и мультиколлинеарность, сформирован итоговый датасет для обучения. Для прогнозирования были обучены и протестированы различные модели регрессии. 

`pandas`, `numpy`, `matplotlib`, `scikit-learn` (`train_test_split`, `GridSearchCV`, `LinearRegression`, `RandomForestRegressor`), `catboost`, `lightgbm`, `pyod (KNN`).

### 5. [Восстановление золота из руды](ml-regression-gold)

**Задача:** Построить модель машинного обучения, которая на основе технологических параметров добычи и очистки предскажет коэффициент восстановления золота из золотосодержащей руды.

Проведен анализ данных с разных этапов обогащения, выявлены аномалии и рассчитана эффективность обогащения по формуле. Выполнена предобработка данных: обработка пропусков, устранение неинформативных признаков. Для обучения использованы случайный лес и решающее дерево. Качество модели оценивалось по метрике sMAPE.

`pandas`, `numpy`, `scikit-learn` (`LinearRegression`, `RandomForestRegressor`, `train_test_split`, `mean_absolute_error`), `catboost`, `lightgbm`, `pyod (KNN)`, `matplotlib`.

### 6. [Выбор локации для скважины](bootstrap-profit)

**Задача:** Построить модель машинного обучения, которая поможет выбрать регион для бурения новой скважины, максимизируя потенциальную прибыль и минимизируя риски убытков.

Предобработка данных и анализ распределений запасов нефти в трех регионах. Для каждого региона обучена модель линейной регрессии для прогнозирования объема запасов. На основе прогнозов смоделирован процесс разведки: из 500 точек отобраны 200 лучших для разработки. Оценена прибыль, а с помощью техники Bootstrap проведен анализ рисков. Выбран регион с наибольшей средней прибылью и вероятностью убытков ниже 2.5%.

`pandas`, `numpy`, `matplotlib`, `scikit-learn` (`LinearRegression`, `train_test_split`, `mean_squared_error`), `scipy`.

### 7. [Анализ рынка видеоигр](games-market-analysis)

**Задача:** На основе исторических данных о продажах, рейтингах и жанрах игр выявить закономерности, определяющие успешность продукта, для планирования рекламной кампании на 2017 год.

Предобработка данных: исправлены типы столбцов, обработаны пропуски в годе выпуска, создан агрегированный столбец all_sales. В ходе исследовательского анализа определены актуальные платформы (PS4, XOne, 3DS, WiiU, PSV). Выявлена умеренная положительная корреляция продаж с оценками критиков. Составлен портрет пользователя для трёх регионов (NA, EU, JP). Проверены гипотезы о равенстве средних пользовательских рейтингов с помощью t-теста.

`pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy` (`ttest_ind`).

### 8. [Исследование надёжности заёмщиков](data-preprocessing-and-analysis)

**Задача:** Определить, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок.

Проведена полная предобработка данных: обработка пропусков, исправление типов, удаление дубликатов. Выполнена лемматизация текстовых данных (цели кредита) для их категоризации. Проведен исследовательский анализ данных (EDA) для выявления ключевых зависимостей.

`pandas`, `numpy`, `pymystem3`.

### 9. [Анализ рынка недвижимости Санкт-Петербурга](real-estate-analysis)

**Задача:** На основе архива объявлений о продаже квартир определить рыночную стоимость объектов недвижимости и выявить параметры, которые на нее влияют.

Проведена комплексная предобработка данных: исправлен разделитель столбцов, обработаны пропуски в 14 столбцах (замена на медианные значения, заполнение на основе коэффициентов из группировки, удаление строк), изменены типы данных. Выполнен исследовательский анализ данных (EDA) для изучения распределений площади, цены, количества комнат и высоты потолков. Добавлены новые столбцы: цена квадратного метра, день/месяц/год публикации, тип этажа. Построены графики для выявления зависимостей цены от различных характеристик квартир.

`pandas`, `matplotlib`, `plotly`, `numpy`.
